# Dori & Ritò: Past Life Pal Quiz

一個可愛的互動式單頁小遊戲，讓狗狗飼主輸入毛孩資訊，隨機抽出「狗狗前世是你的什麼人」的漫畫風結果並可立即分享。

## 使用方式
1. 直接開啟 `index.html`，或在專案資料夾啟動任意靜態伺服器（例如 `python3 -m http.server 8080`）。
2. 按下「我想知道！」並依序完成問題、上傳狗狗照片。
3. 觀賞 5–7 秒的暖心過場動畫後，即可看到結果圖片、手繪輪廓，以及分享按鈕（Facebook / Instagram / 更多社群）。

## 小提醒
- 音樂會在開始測驗後以約 50% 音量播放，若瀏覽器需要互動才能播放，請先點擊按鈕。
- 圖片輪廓採用瀏覽器端的像素邊緣描繪，無需上傳到伺服器。
